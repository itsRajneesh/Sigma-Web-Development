1. Summery of CSS Selector (L-17)

    2. Type of selector

  2. Element selector - div{ }

  3. class selector - .classname{ }

  4. id selector - #idname{ }

  5. Child selector -div>p{ }

  6. descendent selector- div p{ } - select thar element which are child or grandson or descendent of an element

  7. Universal selector -*{ } - select every element

  8. Pseudo selector

  1. a:visited{ } - you can change appearance of visited link by this

  2. a:link{ } - you can change appearance of unvisited link by this

  3. a:active { } - you can change appearance when you click a link

  4. a:hover { } - change appearance when you hover

  5. p:first-child { } - select p or any given element if it is first child of any element
