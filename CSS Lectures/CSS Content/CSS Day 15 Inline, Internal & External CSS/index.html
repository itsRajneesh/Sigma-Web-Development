Inline CSS

Inline CSS is the technique to define the single element with the insert style sheets in an HTML document

Example:

<h3>style="color: white; background-color: orange;">Inline CSS is the technique to define the single element with the insert style sheets in an HTML documen</h3>

Internal CSS

Internal CSS is one of the most widely used CSS forms for changing, styling, and modifying the unique styles of a single web page. You can use the internal CSS by integrating the style element in the head section of a HTML web page

Example: 

                <style>
            body{
                color: purple;
                background-color: yellow;
            }
            </style>

External CSS

External CSS is a form of CSS which is used to add styling to multiple HTML pages at a time. It helps to design the layout of many HTML web pages simultaneously. The external CSS is always saved with the .css extension, and through this file, we can change the complete style of our HTML web page.
